# Hospital Data Analysis Project

## Overview
This project was developed to analyze healthcare data, exploring operational patterns and patient behavior. The analysis focuses on enhancing the understanding of operational efficiency within the hospital and identifying risk factors and groups that require specialized care.

---

## 1. Dashboard Overview
This dashboard consolidates all Key Performance Indicators (KPIs), providing a comprehensive view of hospital activity, patient distribution, the most common diagnoses, and long-stay cases.
![Dashboard Overview](/imges/Dashboard%20Overview.png)

---

## 2. Smoking Impact Analysis
This chart illustrates the relationship between smoking status and patient outcomes (Discharge, DAMA, Expiry). This analysis aids in understanding health risks associated with lifestyle choices and their impact on clinical results.
![Smoking vs Patient Outcome](/imges/Smoking%20Impact%20Analysis.png)

---

## 3. Age Profile Summary
This chart displays the age demographics of patients, helping to identify which groups utilize medical services most frequently and enabling resource planning based on these distributions.
![Age Profile Summary](/imges/Age%20Profile%20Summary.png)

---

## 4. Long-Stay Patients Record
This is a critical component of the operational analysis, where I aggregated detailed data (File Number, Age, Gender, and Duration of Stay) for patients with extended hospitalizations. 
* **Note:** The data reveals that some cases reach up to **98 days** of stay, which assists hospital management in closely monitoring these specific high-attention cases.
![Long-Stay Patients Record](/imges/Long-Stay%20Patients%20Record.png)

---

### Key Technical Skills Demonstrated
* **Data Cleaning & Modeling:** Preparing and transforming raw data into a format suitable for analysis.
* **KPI Development:** Calculating Average Length of Stay (ALOS) and analyzing clinical outcomes.
* **Dashboard Design:** Creating interactive interfaces to support data-driven decision-making.
